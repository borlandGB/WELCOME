🐧 Linux Commands

ls -la
List all files with details
cd /path
Change directory
pwd
Print working directory
cp source dest
Copy files
mv source dest
Move or rename files
rm -rf folder
Remove folder recursively (careful!)
chmod +x script.sh
Make script executable
ps aux | grep process
Find running process
kill -9 PID
Force kill process by ID
df -h
Check disk space
free -h
Check RAM usage
